# 🏀 Quadra Fácil

Sistema simples de agendamento de quadra para condomínios.

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de resolver conflitos de horário no uso da quadra de um condomínio.

A aplicação permite que usuários reservem horários disponíveis e visualizem as reservas já realizadas.

## 🚀 Funcionalidades

- Reserva de horários disponíveis
- Bloqueio de horários já reservados
- Exibição das reservas na tela
- Persistência de dados com localStorage

## 🛠️ Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## ⚠️ Observações

Esta é uma versão inicial (MVP), onde os dados são armazenados localmente no navegador.

Em uma versão futura, o sistema pode ser evoluído para suportar múltiplos usuários com integração a um banco de dados.

## 💡 Melhorias futuras

- Sistema multiusuário
- Integração com backend
- Autenticação de usuários
- Seleção de datas
